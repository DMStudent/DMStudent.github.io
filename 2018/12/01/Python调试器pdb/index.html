Python调试器pdb

pdb是Python自带的Debug工具。

pdb有2种用法：

  1. 非侵入式方法（不用额外修改源代码，在命令行下直接运行就能调试）
    python3 -m pdb filename.py
  2. 侵入式方法（需要在被调试的代码中添加一行代码然后再正常运行代码）
    import pdb;pdb.set_trace() 当你在命令行看到下面这个提示符时，说明已经正确打开了pdb
